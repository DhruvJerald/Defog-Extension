🔹 Installing from a .zip file
Download the repository from GitHub (usually via the green Code → Download ZIP button).

Extract the ZIP to a folder on your computer.

Open Chrome and go to chrome://extensions/.

Toggle Developer mode (top right).

Click Load unpacked.

Select the folder that contains the manifest.json file (the root of the extension).

The extension will now appear in your list.

🔹 Installing from a .crx file
Download the .crx file from GitHub (sometimes found under Releases).

Open Chrome and go to chrome://extensions/.

Toggle Developer mode.

Drag and drop the .crx file into the extensions page.

Confirm the installation when prompted.

⚠️ Notes
If Chrome blocks .crx installation, you can still use the Load unpacked method with the source code.

Always make sure the extension is from a trusted source before installing.
